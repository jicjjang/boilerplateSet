# Front Framework

## 1. Descrption

Boilerplate Spring boot with Vuejs using Intellij.

## 2. Install

~~~shell
$ git clone https://github.com/jicjjang/boilerplateSet.git
$ cd boilerplateSet/frontFramework/frontend
$ npm i
$ npm run build
~~~

## 3. Boot Intellij

1. File > Open...
2. Run > Run 'VueApplication'
